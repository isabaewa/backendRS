Описание проекта:

Это серверная часть проекта кафе - Roll & Soul, реализованная на Python (Flask).
Сервер обрабатывает меню и бронирования клиентов, предоставляет REST API для фронтенда (Next.js).

Структура проекта:
project/
│
├── backend/
│   ├── app.py            # Flask сервер
│   ├── menu.json         # Меню блюд
│   ├── bookings.json     # Список бронирований
│   └── README.md         # Описание проекта (этот файл)
│
└── venv/                 # Виртуальное окружение Python

## Маршруты

### GET /menu
Возвращает список блюд из файла `menu.json`.

### POST /book
Принимает бронирование и сохраняет его в `bookings.json`.

**Пример JSON-запроса:**
```json
{
  "name": "Алина",
  "phone": "8700...",
  "date": "2025-10-13",
  "time": "18:30",
  "guests": 3
}
Пример ответа:

{"message": "Бронирование успешно сохранено"}

GET /bookings

Возвращает все бронирования (для проверки и тестирования).
